<h1><a href="">문자열 다루기 기본</a></h1>

<h6>문제 설명</h6>
<p>문자열 s의 길이가 4 혹은 6이고, 숫자로만 구성돼있는지 확인해주는 함수, solution을 완성하세요. 예를 들어 s가 <q>a234</q>이면 False를 리턴하고 <q>1234</q>라면 True를 리턴하면 됩니다.</p>

<h5>제한 사항</h5>

<ul>
<li><code>s</code>는 길이 1 이상, 길이 8 이하인 문자열입니다.</li>
</ul>

<h5>입출력 예</h5>

        <thead><tr>
<th>s</th>
<th>return</th>
</tr>
</thead>
        <tbody><tr>
<td><q>a234</q></td>
<td>false</td>
</tr>
<tr>
<td><q>1234</q></td>
<td>true</td>
</tr>
</tbody>
</table>
